# JavaScript-Fundamentals-Problem-Based-Solutions
Making application in Person-Problem #6
This application will help users identify its 
subscriptions total monthly burn whether is was 
used or not.

---Customer Profile---
Jobs
1. Track monthly subscriptions in one place
2. Avoid forgetting payment due dates
3. Manage entertainment and app expenses
4. Know where money is being spent
Pains
1. Forgetting active subscriptions
2. Unexpected auto-renewal charges
3. Too many apps charging at different dates
4. Difficult to calculate total monthly expenses
Gains
1. Clear overview of all subscriptions
2. Better budgeting and expense control
3. Payment reminders before due dates
4. Ability to cancel unused services
---Value Proposition---
 Products and Services
1. Subscription management web/mobile app
2. Monthly expense calculator
3. Renewal reminders and notifications
4. Subscription category organizer
5. Spending summary dashboard
Pain Relievers
1. Centralized list prevents forgotten subscriptions
2. Notifications reduce late or surprise charges
3. Expense tracking simplifies budgeting
4. Organized dashboard reduces confusion
Gain Creators
1. Users save money by spotting unused services
2. Visual charts improve spending awareness
3. Reminders create peace of mind
4. Budget insights help users make smarter financial decisions
